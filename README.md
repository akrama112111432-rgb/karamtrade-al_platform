# منصة KaramTrade — مشروع تجريبي

مرحباً — هذا المستودع يحتوي على نسخة واجهة عرض (Landing) عربية وتجريبية لمنصة تداول اسمها **KaramTrade**، مع ملفات جاهزة للواجهة (HTML/CSS/JS) وإرشادات مبسطة لبناء Backend ونشر الموقع.

## نظرة عامة
- الهدف: صفحة عرض احترافية باللغة العربية + مكونات تجريبية (تسجيل، محاكاة صفقات، دردشة تجريبية، صفحة "أرباحي"، ومسابقات).
- مخصصة للعرض والتعلّم. لميزات حقيقية (تداول بأموال، بوابات دفع) تحتاج إعدادات backend واشتراك في خدمات دفع.

## بنية المجلد (أهم الملفات)
- `index.html` — الصفحة الرئيسية (عربية، RTL).
- `style.css` — أنماط التصميم.
- `app.js` — سكربتات التفاعل (تنبيهات، محاكاة صفقات).
- `README.md` — هذا الملف.
- `backend/` — دليل وتعليمات لملفات الخادم (مثال Node.js).
- `backend/.env.example` — نموذج متغيرات بيئة.
- `monetization.md` — أفكار تحقيق أرباح.
- `contest_rules.md` — نموذج قوانين المسابقات.
- `LICENSE` — رخصة MIT.

## كيف تشغّل الواجهة محليًا (سهل)
1. ضع كل الملفات في مجلد واحد على جهازك.
2. افتح `index.html` في متصفح هاتفك أو الكمبيوتر (نقرة مزدوجة أو "فتح بواسطة المتصفح").
3. ستشاهد الواجهة التجريبية، يمكن تجربة الأزرار والتفاعلات فوراً.

## نشر مباشرة على GitHub Pages
1. قم بعمل مستودع جديد على GitHub (أو استخدم المستودع الحالي).
2. ارفع كل الملفات إلى الفرع `main` (أو `master` إن كنت تستخدمه).
3. اذهب إلى إعدادات المستودع (Settings) → Pages.
4. اختَر مصدر النشر: **Branch** = `main` و **Root** = `/` ثم احفظ (Save).
5. بعد دقيقة-دقيقتين افتح: `https://USERNAME.github.io/REPO_NAME/` (استبدل USERNAME وREPO_NAME).
